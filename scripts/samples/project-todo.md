一个用 Python 写的命令行待办事项管理工具。

## 功能

- 添加、完成、删除待办事项
- 标签分类
- 优先级标记
- 全文搜索
- 数据持久化（JSON 文件）

## 用法

```bash
todo add "写博客" --priority high --tag writing
todo list
todo done 1
```
