这是我的个人站点第一篇文章。

站点使用纯 HTML/CSS/JS 构建，零依赖部署在 GitHub Pages 上。从设计到实现，从樱花粒子到隐藏彩蛋，每行代码都是亲手写的。

## 为什么自己写？

用 WordPress、Hexo 这类工具当然更快，但少了很多乐趣。自己写可以：

- 完全掌控每一像素
- 想加什么彩蛋就加什么
- 学习底层原理
